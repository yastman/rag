### 2. Оптимизация BGE-M3 Memory (СРЕДНИЙ ПРИОРИТЕТ) ⭐⭐

**Проблема:** BGE-M3 использует 2.048 GB / 3 GB (68% загрузка).

**Решение:**
```python
# В config.py добавить параметры для FlagEmbedding
BGE_M3_BATCH_SIZE = 32  # Уменьшить с 64 (если используется)
BGE_M3_MAX_LENGTH = 512  # Ограничить длину текста
BGE_M3_NORMALIZE = True  # Нормализовать векторы для экономии памяти
```

**Альтернатива:** Использовать квантизацию модели (FP16 → INT8).

**Ожидаемый эффект:**
- ✅ Снижение памяти на 20-30% (до ~1.5 GB)
- ✅ Небольшое ускорение инференса (~10-15%)
- ⚠️ Минимальное снижение качества (<0.5%)

**Сложность:** Средняя
**Время:** 2-3 часа (тестирование качества)
**ROI:** 🔥 Средний

async def embed_batch(texts: list[str], batch_size=32):
