# Plans

This directory holds dated design notes, implementation plans, audit plans, and
triage snapshots. Treat files here as planning history or evidence unless a
current owner doc links one as active.

Use the canonical docs for current rules and source-of-truth guidance:

- Project docs navigation: [`../README.md`](../README.md)
- SDK/framework guidance: [`../engineering/sdk-registry.md`](../engineering/sdk-registry.md)
- Operational runbooks: [`../runbooks/README.md`](../runbooks/README.md)

When adding a plan, include the date in the filename and keep acceptance
criteria, verification, and rollback notes in the plan itself.
