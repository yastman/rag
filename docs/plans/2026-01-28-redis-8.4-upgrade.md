# Redis 8.4 Upgrade Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Upgrade Redis from `redis/redis-stack:latest` to `redis:8.4.0` for stability and performance.

**Architecture:** Replace unpinned Redis Stack image with pinned Redis 8.4 OSS. Redis 8.4 includes Query Engine (FT.* commands) natively. Add smoke test to verify Query Engine availability.

**Tech Stack:** Redis 8.4.0, Docker Compose, Make, pytest

## Task 1: Add Redis Smoke Test to Makefile

**Files:**
- Modify: `Makefile:153` (after `test-all-smoke-load`)

**Step 1: Write the test target**

Add after line 154 in Makefile:

```makefile
# =============================================================================
# REDIS VERIFICATION
# =============================================================================

.PHONY: test-redis

test-redis: ## Verify Redis Query Engine is available
	@echo "$(BLUE)Testing Redis Query Engine...$(NC)"
	@docker exec dev-redis redis-cli FT._LIST > /dev/null 2>&1 || \
		(echo "$(RED)FAIL: FT._LIST not available - Query Engine missing$(NC)" && exit 1)
	@docker exec dev-redis redis-cli FT.CREATE __test_idx ON HASH PREFIX 1 __test: SCHEMA name TEXT > /dev/null 2>&1 || \
		(echo "$(RED)FAIL: Cannot create test index$(NC)" && exit 1)
	@docker exec dev-redis redis-cli FT.DROPINDEX __test_idx > /dev/null 2>&1 || true
	@echo "$(GREEN)✓ Redis Query Engine OK$(NC)"
```

**Step 2: Run to verify syntax**

Run: `make help | grep redis`

Expected:
```
  test-redis           Verify Redis Query Engine is available
```

**Step 3: Commit**

```bash
git add Makefile
git commit -m "feat(redis): add Query Engine smoke test

Co-Authored-By: Claude Opus 4.5 <noreply@anthropic.com>"
```
