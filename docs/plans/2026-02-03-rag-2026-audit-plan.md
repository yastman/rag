## 1. Overview

### 1.1 Что проверяем

| Wave | Milestones | Scope |
|------|------------|-------|
| 1 (P0) | A, A.0, A.2 | Infrastructure, docs consolidation, test data |
| 2 (P1) | A.1, B, C, F, I | Langfuse, binary quantization, small-to-big, RAG eval, alerting |
| 3 (P2) | D, G, H, J | ACORN, HyDE, guardrails, ingestion pipeline |
| 4 (R&D) | E | Contextualized embeddings (voyage-context-3) |

### 1.2 Типы проверок

1. **Verification** — код/тесты/файлы существуют и работают
2. **Completeness** — все scope items из плана выполнены
3. **Integration** — части работают вместе end-to-end

### 1.3 Критерии успеха

- [ ] `make test` — все тесты проходят
- [ ] `make check` — lint + types clean
- [ ] Qdrant collections с данными
- [ ] Docker сервисы работают
- [ ] Feature flags функционируют
- [ ] Bot отвечает на запросы
