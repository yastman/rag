```yaml

  cause: Incomplete requirements.txt
  solution: Check pyproject.toml, install missing package
  task: Task 1.3 will fix this

"requests blocking event loop":
  cause: Sync requests in async context
  solution: Replace with httpx.AsyncClient
  task: Task 1.2 will fix this

"Out of memory":
  cause: Multiple embedding models loaded
  solution: Implement singleton pattern
  task: Task 2.1 will fix this

"GitHub Actions not running":
  check: Settings → Actions → Enabled?
  check: Workflow syntax correct?
  debug: gh workflow list && gh run list

"Tests failing":
  run: pytest tests/ -v -s
  check: Recent changes
  fix: Revert or fix issue
```
